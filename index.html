Minutes per player (all 5 games)

Each half = 12:30. Red = 0 min that game. Orange = total < 50 min. Green = ≥ 60 min. GK = goalkeeper that game (full game); GK·H1 / GK·H2 = goalkeeper one half only. Above the games: player pills use a continuous red-yellow-green gradient based on % of fair share (deep red < 50%, yellow = on target, deep green > 140%). Pills are sorted ascending by minutes so players needing more time appear first. The percentage next to each player is their current share vs. expected. Click any pill (or chip) to highlight every appearance of that player across all games. Click more pills to add to the selection. Click a highlighted pill again to remove it, or use the Clear button. A small ⚙ gear appears on the right of each position's label — click it to reveal merge/split buttons for that spot. Merged positions get Split; divided positions get Keep [starter] / Keep [sub]. A player benched both halves of a game (not playing) is highlighted in red on both of their bench chips inside the affected game. On the pitch, a chip gets a red border if its player is in a divided position here but is also on the field in the other half elsewhere (i.e. the "sub" is actually just a position switch, not a real substitution). Use the Versions ▾ button to save the current plan under a name and reload it later; each game's current state is always auto-saved separately from named versions. Drag any chip to any slot within the same game to reorganize.
